# Healthcare Data Analytics Project

## Project Overview

This project focuses on analyzing healthcare insurance data to understand the factors influencing medical costs.

The dataset contains patient demographic information such as age, gender, BMI, smoking status, and geographic region. Using data analytics techniques, we analyze how these factors affect healthcare expenses.

The project includes data cleaning, exploratory data analysis, SQL-based analytics, and an interactive Power BI dashboard.

## Objectives

- Analyze healthcare cost patterns
- Identify factors affecting medical expenses
- Study the impact of smoking on healthcare costs
- Analyze BMI influence on treatment costs
- Compare medical costs across regions
- Build a dashboard for healthcare insights
