# Dataset Description

## Dataset Name
Medical Cost Personal Dataset

## Source
Kaggle Healthcare Insurance Dataset

## Dataset Size
Total Records: 1338  
Total Columns: 7

## Column Description

### age
Age of the patient.

### sex
Gender of the patient (Male / Female).

### bmi
Body Mass Index of the patient.

### children
Number of dependents covered by insurance.

### smoker
Indicates whether the patient is a smoker (Yes / No).

### region
Patient's residential region:
- northeast
- northwest
- southeast
- southwest

### charges
Medical insurance cost billed to the patient.
