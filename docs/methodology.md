# Methodology

## Phase 1 – Data Collection
The healthcare insurance dataset was collected from Kaggle. The dataset contains patient demographic information and medical insurance charges.

## Phase 2 – Data Cleaning
Data preprocessing was performed using Python and Pandas.

Steps performed:
- Checked for missing values
- Removed duplicate records
- Verified dataset structure
- Exported cleaned dataset

## Phase 3 – Feature Engineering
New features were created to improve analysis.

Age Groups
- Young
- Middle Age
- Senior

BMI Categories
- Underweight
- Normal
- Overweight
- Obese

Risk Level
- Low Risk
- Medium Risk
- High Risk

## Phase 4 – Exploratory Data Analysis
EDA was performed using Python and Matplotlib to understand patterns in the dataset.

Key analysis included:
- Age distribution
- Gender distribution
- Medical cost distribution
- Smoking impact on medical costs
- BMI impact on healthcare costs
- Region-wise patient distribution

## Phase 5 – SQL Analytics & Power BI Dashboard
The cleaned dataset was loaded into a MySQL database to perform analytical queries.

SQL analysis included:
- Total healthcare revenue
- Average treatment cost
- Region-wise revenue analysis
- Risk level cost analysis

An interactive Power BI dashboard was created to visualize key healthcare insights including:
- Total revenue
- Average medical cost
- Smoking cost comparison
- Age group analysis
- BMI category analysis
- Regional revenue distribution
